Two Phase DDoS Mitigation

Mineflex is protected from volumetric DDoS attacks at the network edge.

Network Edge Mitigation

Volumetric attacks are halted at the network edge in our first phase of mitigation. This helps absorb attacks that could overwhelm our capacity to the public internet.

Mineflex TCP Load Balancer

As a phase two mitigation no Mineflex Minecraft Server is exposed to the public internet, all connections must pass through our TCP Load Balanced cluster.

IP Firewall

Mineflex allows you to block ip connections to your server at our Load Balancers. This provides much greater capacity and flexibity than letting your Minecraft Server apply IP filtering.

Deep Packet Inspection

We can mitigate more sophisticated attacks here by performing deep packet inspection and ensuring all packets are valid minecraft protocol packets.
